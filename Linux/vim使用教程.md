**Vim进入和退出命令**

1. :w 保存文件但不**退出**vi.
2. :w file 将修改另外保存到file中，不**退出**vi.
3. :w! 强制保存，不推出vi.
4. :wq 保存文件并**退出**vi.
5. :wq! 强制保存文件，并**退出**vi.
6. :q 不保存文件，**退出**vi.
7. :q! 不保存文件，强制**退出**vi.
8. :e! 放弃所有修改，从上次保存文件开始再编辑
